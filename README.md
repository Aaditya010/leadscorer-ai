# LeadScorer AI
A smart sales dashboard that uses AI to score leads (hot vs cold) based on website behavior.

## Tech Stack
- **Backend:** FastAPI (Python)
- **Database:** PostgreSQL (Supabase)
- **Frontend:** React

## Features
- ✅ Add leads via API
- ✅ Fetch all leads from database

- 🚧 ML scoring -->soon
